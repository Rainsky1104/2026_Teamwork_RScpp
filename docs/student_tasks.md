# 学生任务说明

## 课程设计目标

3-4 人一组，完成一个工程结构合理、可运行、可扩展的遥感影像处理系统。评分不仅看功能数量，也看模块划分、接口设计、异常处理、代码规范和团队协作。

## 基础功能

- 工程图层：左侧树状结构按“源数据/遥感影像、点云、Mesh、DEM”和“处理结果/直方图、均衡化、分类”等分组显示。
- 遥感影像加载：使用 GDAL 读取 GeoTIFF/IMG/JP2 等数据，显示宽高、波段数、投影和地理变换。
- 波段显示：影像节点下展开 Band 1..N；选中影像显示 RGB 合成，选中波段显示单波段灰度。
- 浏览交互：二维影像支持滚轮缩放、鼠标拖动、适应窗口；三维视图支持旋转、平移、缩放。
- 影像处理：至少实现 3 个参数可配置算法，例如直方图、均衡化、阈值分割、边缘检测、监督/非监督分类、目标检测。
- 日志与错误处理：所有加载、处理、导出操作都应写入日志；失败时给出明确错误信息。

## 摄影测量与三维方向

- DEM 重建：选中两张影像，弹出参数对话框，选择相机参数、控制点或输出目录，自动完成特征提取、匹配、校正、视差和 DEM 输出。
- 正射校正：选中影像和其对应 DEM，根据地理关系或相机模型重采样生成正射影像。
- 三维展示：加载 DEM/点云/Mesh，将影像贴到 DEM 上展示，支持基础交互。

## 进阶功能

- 多波段指数：NDVI、NDWI、NDBI 等。
- 分类：K-Means、SVM、Random Forest 或深度学习模型调用。
- 特征方法对比：ORB/SIFT/AKAZE 的匹配数量、内点率和耗时统计。
- 数据导出：GeoTIFF、CSV、PLY、OBJ。
- 工程保存与恢复：保存图层列表、参数、结果路径。
- 安装包：使用 windeployqt、NSIS、Inno Setup 或 CPack 打包。
- 单元测试：对算法核心函数添加测试数据与测试用例。

## 必须体现的 C++ 知识点

- 继承与多态：所有处理算法继承统一的 `ProcessingAlgorithm`。
- 模板：使用或扩展 `LayerManager<T>` 管理不同数据对象。
- 智能指针：图层和算法对象避免裸指针所有权混乱。
- STL/Qt 容器：合理使用 `std::vector`、`QVector`、`QStringList` 等。
- 异常处理：文件读取、参数错误、算法失败必须捕获并记录。
- 模块化：IO、数据模型、算法、UI、三维显示分目录实现。

## 提交材料

- 可编译运行的 Qt/CMake 工程。
- 设计文档：功能结构、类图、关键接口说明、团队分工。
- 测试文档：测试数据来源、操作步骤、截图、结果分析。
- 演示视频或现场演示。
